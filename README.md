# CSS 渐变生成器

> CSS 渐变可视化编辑器，线性/径向渐变实时预览，一键复制标准 CSS

## 功能概览

- **线性渐变（linear-gradient）角度 0-360° 滑块调节**
- **径向渐变（radial-gradient）椭圆/圆形 + 9 种起始位置**
- **最多 8 个色标节点，取色器调色 + 滑块调位置**
- **12 组精美预设模板（日落/海洋/极光/森林/星空等）**
- **输出兼容 -webkit- / -moz- 前缀的生产级 CSS**

## 操作说明

1. 打开 index.html（浏览器直接运行，无需服务器）
2. 按界面引导输入/操作
3. 结果实时预览，支持一键复制导出

## 技术栈

- **HTML5**：语义化结构
- **CSS3**：白色极简主题，Flex/Grid 布局
- **Vanilla JavaScript**：纯原生代码，零依赖
- **ES6+**：sync/await、箭头函数、模板字符串
- **localStorage**：偏好设置持久化
